# Användarguide – byta till den nya Microsoft 365-miljön (mobil)

**Gäller iPhone/iPad och Android.** Du behöver din **nya e-postadress** och ditt **tillfälliga
lösenord**.

## iPhone / iPad

### Ta bort det gamla kontot
1. Inställningar → **Appar** → **Mail** → Konton (eller Inställningar → Kontakter → Konton).
2. Välj det gamla arbetskontot → **Radera konto**.
3. Har du Microsoft Outlook-appen: öppna den, gå till inställningar (kugghjul) → välj gamla
   kontot → **Ta bort konto**.

### Lägg till det nya kontot
1. Installera/öppna **Microsoft Outlook** från App Store.
2. Lägg till konto → ange din **nya e-postadress** → det **tillfälliga lösenordet** → godkänn MFA.
3. Installera/öppna även **Teams** och **OneDrive** och logga in med det nya kontot.

## Android

### Ta bort det gamla kontot
1. Inställningar → **Konton** (eller *Lösenord och konton*) → välj det gamla arbetskontot →
   **Ta bort konto**.
2. I Outlook-appen: kugghjul → gamla kontot → **Ta bort konto**.

### Lägg till det nya kontot
1. Öppna **Microsoft Outlook** → Lägg till konto → din **nya e-postadress** → **tillfälligt
   lösenord** → godkänn MFA.
2. Logga in i **Teams** och **OneDrive** med det nya kontot.

> **Tips.** Om appen fortfarande visar gammal e-post: stäng appen helt och öppna igen, eller
> starta om telefonen. Kvarstår problem, kontakta IT-supporten.
